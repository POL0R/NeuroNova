# Final App Media Files

This folder contains photos and videos of the final NeuroNova application.

## Folder Structure

### Photos
Place final app photos in: `final-app/photos/`

Suggested naming:
- `app-screenshot-home.png`
- `app-screenshot-module-stress-bridge.png`
- `app-screenshot-module-sequence-maze.png`
- `app-screenshot-module-color-tiles.png`
- `app-screenshot-module-shop-market.png`
- `app-screenshot-module-emotion-match.png`
- `app-screenshot-module-routine-builder.png`
- `app-screenshot-classroom-use.png`
- `app-screenshot-teacher-interface.png`

### Videos
Place final app videos in: `final-app/videos/`

Suggested naming:
- `app-demo-full.mp4`
- `app-demo-module-stress-bridge.mp4`
- `app-demo-module-sequence-maze.mp4`
- `app-demo-module-color-tiles.mp4`
- `app-demo-module-shop-market.mp4`
- `app-demo-module-emotion-match.mp4`
- `app-demo-module-routine-builder.mp4`
- `app-demo-classroom-usage.mp4`
- `app-demo-teacher-guide.mp4`
- `app-demo-offline-feature.mp4`

## File Formats
- **Photos**: PNG, JPG, JPEG, WebP
- **Videos**: MP4, WebM, MOV

## Usage
These files can be used on the website to showcase:
- Final app screenshots
- Module demonstrations
- Classroom usage examples
- Teacher interface
- User testimonials
- App Store previews
