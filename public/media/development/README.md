# Development Media Files

This folder contains photos and videos from the development process of NeuroNova.

## Folder Structure

### Photos
Place development photos in: `development/photos/`

Suggested naming:
- `development-screenshot-01.png`
- `development-ui-design.png`
- `development-team-working.png`
- `development-prototype.png`

### Videos
Place development videos in: `development/videos/`

Suggested naming:
- `development-process-01.mp4`
- `development-demo-early.mp4`
- `development-testing.mp4`

## File Formats
- **Photos**: PNG, JPG, JPEG, WebP
- **Videos**: MP4, WebM, MOV

## Usage
These files can be used on the website to showcase:
- Development journey
- Behind-the-scenes content
- Team collaboration
- Prototype iterations
- Testing phases
