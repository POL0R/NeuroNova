## File Format Notes:
- All files should be in PDF format
- File names use lowercase with hyphens for readability
- Format: `[number]-[organization/name]-[key-person]-[location]-[date].pdf`
- Dates follow YYYY-MM-DD format for chronological sorting
- Numbers at the start ensure proper ordering
